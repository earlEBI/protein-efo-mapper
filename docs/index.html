pQTL Measurement Mapper

Maps UniProt accessions, gene symbols, gene IDs, and protein names to measurement terms in EFO and OBA-imported EFO terms.

EFO + OBA Matrix-aware (plasma > blood > serum) Validated vs review-queue outputs

Input Template

Use one query column. Add input_type when known for better precision.

query	input_type
Q9ULI3	accession
LIPC	gene_symbol
ENSG00000172137	gene_id
Beta-glucosidase 2	protein_name
Download TSV Template

Run Mapper

Run locally from your project root:

.venv/bin/python skills/pqtl-measurement-mapper/scripts/map_measurement_efo.py map \
  --input data/analytes.tsv \
  --output final_output/analytes_efo.tsv \
  --review-output final_output/review_queue.tsv \
  --index skills/pqtl-measurement-mapper/references/measurement_index.json \
  --workers 8 \
  --parallel-mode process \
  --progress

Outputs

  • final_output/analytes_efo.tsv: main mappings with confidence and validation.
  • final_output/review_queue.tsv: unresolved or withheld mappings with top suggestions.
  • final_output/analytes_unmapped.tsv: optional pure unresolved list.
